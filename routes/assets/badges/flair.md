# Flair Badge
This endpoint returns a PNG of a flair badge. The flair name can be gotten from `donation[flair]`.

## Endpoint
`GET` [/assets/badges/\<flair\>](https://assets01.teamassets.net/assets/badges/feed-icon-1.png)
